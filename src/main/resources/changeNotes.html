v1.6
  • Optimize the table structure modification. Now every generation will delete the deleted columns and modify the existing columns
  • Add. Define annotation, support custom column definition, and the highest priority can override JDBC_ TYPE\LEN\NO_ Null / default / desc comment
  • Modify part of the JDBC type to database type mapping
  • The. Key annotation can be modified to appendable incr attribute, which indicates that the database is auto incremented
  • The core execution API allows customization of entitybuilder, mapperbuilder, and xmlbuilder
  • Add SQL execution log, which is saved in SQL folder under XML directory by default
v1.5
  • Fix: unable to read the disk file in memory. The edited file can be read even if it is not written to the disk
  • Optimization: add the function of refreshing the disk to modify the file, which can be viewed immediately after the file is generated
  • Optimization: added rebuild table function
  • Optimization: added tick configuration function
  • Optimization: support each project to save configuration file separately
  • Optimization: generator optimization / add some functions, please see the description file for details https://github.com/llongtao/mybatis-helper
  • Fix: character set problem, output in UTF-8 by default
